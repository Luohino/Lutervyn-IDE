# Lutervyn IDE Changes Log

## [1.0.0] - 2026-01-29

Initial release of Lutervyn IDE, an AI-powered lightweight text editor.

### Features

* AI-powered code assistance
* Lightweight and fast text editing
* Support for multiple programming languages
* Plugin system for extensibility
* High DPI display support
* Retina display support on macOS

### Fixes

* Initial setup and configuration

### Other Changes

* Rebranded from Lite XL to Lutervyn IDE
